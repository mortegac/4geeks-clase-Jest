## Instalar proyecto actual

- Clonar repositorio
```
https://github.com/mortegac/4geeks-clase-Jest.git
```

- Instalar dependencia por la terminal
```
npm install
```

## Comandos

- Ejecutar test
```
npm run test
```

- Ejecutar coverage
```
npm run coverage
```


## Iniciar un proyecto vacío

- Iniciar él proyecto
```
npm init --y
```
Instalación de Jest
```
npm install jest
```
